PARTE I — WORD 365

1. CONCEPTO Y FUNCIONES PRINCIPALES DE WORD 365

1.1 Qué es Word 365

Microsoft Word es el procesador de texto más utilizado del mundo, integrado en el paquete Microsoft 365 (anteriormente Office). Un procesador de texto es una aplicación que permite crear, editar, dar formato, gestionar e imprimir documentos de texto con plenas capacidades tipográficas y de diseño.

Word 365 es la versión de suscripción de Word, que se actualiza continuamente con nuevas funciones. Sus funcionalidades son equivalentes a las de Word 2021, con la ventaja de recibir mejoras periódicas automáticas y permitir la colaboración en línea a través de OneDrive y SharePoint.

2. CREACIÓN Y ESTRUCTURACIÓN DEL DOCUMENTO

2.1 Creación de un documento

Al abrir Word se muestra la pantalla de inicio, desde la que se puede:

  • Crear un documento en blanco (Ctrl + U).
  • Crear un documento a partir de una plantilla (template): documentos preformatados para cartas, currículums, informes, facturas, etc.
  • Abrir un documento reciente.

2.4 Estilos

Los estilos son conjuntos de formatos predefinidos (fuente, tamaño, alineación, interlineado, etc.) que se aplican de una sola vez a un texto. Son el elemento más importante para estructurar un documento de forma coherente y eficiente.

Los estilos se encuentran en Inicio → grupo Estilos. Los más importantes son:

  • Normal: estilo del cuerpo del documento; el texto sin estilo específico.
  • Título 1, Título 2, Título 3…: estilos de encabezado jerárquicos. Su uso es fundamental para que Word pueda generar automáticamente la tabla de contenido y para navegar por el documento mediante el Panel de navegación.
  • Énfasis, Énfasis intenso: variantes de cursiva o negrita para destacar texto.
  • Cita: para citas bibliográficas o bloques de texto especiales.

El uso correcto de estilos permite:

  • Generar la tabla de contenido automáticamente.
  • Mantener la consistencia visual de todo el documento.
  • Modificar el formato de todas las instancias de un estilo de forma global con un solo cambio.

2.5 Encabezados y pies de página

Los encabezados y pies de página son zonas que aparecen en la parte superior e inferior de cada página del documento, fuera del área de texto principal. Se insertan desde Insertar → Encabezado / Pie de página.

Permiten incluir: número de página, título del documento, nombre del autor, fecha, logotipo institucional, etc.

Los encabezados y pies de página son independientes del área de texto; para editar el documento hay que hacer doble clic fuera del área de encabezado/pie para salir del modo de edición.

Se puede configurar encabezado/pie diferente para la primera página y encabezados/pies distintos para páginas pares e impares desde Diseño del encabezado y pie de página → opciones.

2.6 Número de página

Los números de página se insertan desde Insertar → Número de página, con opciones de posición (arriba, abajo, al margen) y formato (1, 2, 3… / i, ii, iii… / a, b, c…). Pueden iniciarse desde cualquier número.

2.7 Columnas

El texto de un documento puede organizarse en columnas (como en un periódico) desde Disposición → Columnas. Puede aplicarse al documento completo o solo a una sección seleccionada.

2.8 Tablas

Las tablas son cuadrículas de filas y columnas para organizar información. Se insertan desde Insertar → Tabla. Una vez creada:

  • Se pueden añadir/eliminar filas y columnas con el menú contextual o las fichas Diseño de tabla y Disposición (que aparecen al seleccionar la tabla).
  • Se puede combinar o dividir celdas.
  • Se pueden aplicar estilos de tabla predefinidos.
  • Se puede ajustar el ancho de columnas arrastrando los bordes.

2.9 Imágenes e ilustraciones

Desde Insertar → Imágenes se puede insertar imágenes desde el equipo o desde internet (Imágenes en línea). También pueden insertarse formas, iconos, modelos 3D, SmartArt (diagramas) y gráficos.

Una vez insertada una imagen, la ficha Formato de imagen permite ajustar el brillo, contraste, recorte, estilos de imagen y la posición respecto al texto (disposición del texto: en línea, cuadrado, estrecho, delante del texto, etc.).

2.10 Tabla de contenido

La tabla de contenido (índice) se genera automáticamente desde Referencias → Tabla de contenido, basándose en los estilos de Título aplicados al documento. Se actualiza con clic derecho → Actualizar campo o con la tecla F9.

2.11 Notas al pie y notas al final

Las notas al pie se insertan al pie de la página actual; las notas al final, al final del documento. Se insertan desde Referencias → Insertar nota al pie (Ctrl + Alt + F) o Insertar nota al final. Son útiles para referencias, aclaraciones o bibliografía.

2.12 Revisión ortográfica y gramatical

Word revisa automáticamente la ortografía y gramática mientras se escribe, subrayando en rojo los errores ortográficos y en azul los gramaticales. La revisión completa se realiza desde Revisar → Ortografía y gramática (F7).

El idioma de revisión puede cambiarse desde Revisar → Idioma → Establecer idioma de revisión.

2.13 Seguimiento de cambios y comentarios

  • Seguimiento de cambios (Ctrl + Mayúsc + E): registra todas las modificaciones realizadas en el documento, mostrando en color qué texto se ha añadido o eliminado. Es útil para revisión colaborativa de documentos.
  • Comentarios (Ctrl + Alt + M): permite añadir notas marginales sin modificar el texto del documento. Son visibles en el margen derecho.

Ambas funciones están en la ficha Revisar.

2.14 Combinar correspondencia

La combinación de correspondencia (ficha Correspondencia) permite generar documentos personalizados en serie (cartas, etiquetas, sobres, correos electrónicos) a partir de un documento base y una fuente de datos (lista de destinatarios en Excel, Access, contactos de Outlook, etc.).

Pasos básicos:

  1. Crear el documento base en Word.
  2. Seleccionar la fuente de datos (lista de destinatarios).
  3. Insertar campos de combinación en el documento (Nombre, Dirección…).
  4. Vista previa de los resultados.
  5. Completar la combinación (imprimir o generar documentos individuales).

3.2 Recuperación de documentos

Si Word se cierra inesperadamente sin guardar, ofrece la recuperación de documentos al reiniciarse, mostrando las versiones autoguardadas disponibles.

La frecuencia del autoguardado se configura en Archivo → Opciones → Guardar → Guardar información de Autorrecuperación cada X minutos.

Word 365 también mantiene un historial de versiones cuando el documento está guardado en OneDrive o SharePoint, permitiendo recuperar versiones anteriores del documento.

3.3 Impresión de documentos

La impresión se gestiona desde Archivo → Imprimir (Ctrl + P), que muestra la vista previa e impresión con las siguientes opciones:

  • Impresora: selección de la impresora a utilizar.
  • Número de copias.
  • Intervalo de páginas: todas las páginas, página actual, selección, páginas concretas (ej. 1-3, 5, 7-9).
  • Impresión a una cara / a doble cara (si la impresora lo admite).
  • Intercalar: si se imprimen varias copias, imprime el documento completo antes de comenzar la siguiente copia (en lugar de varias copias de la página 1, luego de la página 2, etc.).
  • Orientación: vertical u horizontal.
  • Tamaño del papel.
  • Márgenes.
  • Páginas por hoja: permite imprimir 2, 4, 6, 8 páginas en una sola hoja.

4. PERSONALIZACIÓN DEL ENTORNO DE TRABAJO

4.1 Opciones de Word

Las opciones de configuración de Word se encuentran en Archivo → Opciones, organizadas en categorías:

  • General: nombre de usuario, tema de Office (claro, oscuro, de colores…), pantalla de inicio.
  • Pantalla: opciones de visualización del documento (regla, barras de desplazamiento, señales de formato…).
  • Revisión: configuración de la autocorrección, revisión ortográfica y gramatical.
  • Guardar: formato de guardado predeterminado, ubicación predeterminada, configuración del autoguardado y la autorrecuperación.
  • Idioma: idiomas de edición y visualización.
  • Accesibilidad: opciones para personas con discapacidad visual o motriz.
  • Avanzado: opciones de edición, corte y pegado, visualización, impresión, etc.
  • Personalizar cinta de opciones: añadir, eliminar o reorganizar fichas y grupos de comandos en la cinta.
  • Barra de herramientas de acceso rápido: añadir o eliminar comandos de la barra de acceso rápido.

5. CONCEPTO Y FUNCIONES PRINCIPALES DE EXCEL 365

5.1 Qué es Excel 365

Microsoft Excel es la hoja de cálculo más utilizada del mundo, integrada en Microsoft 365. Una hoja de cálculo es una aplicación que permite organizar datos en formato de tabla (filas y columnas), realizar cálculos mediante fórmulas y funciones, y visualizar los datos mediante gráficos.

Excel es la herramienta fundamental para la gestión de datos numéricos, presupuestos, estadísticas, listas, seguimiento de gastos y cualquier tarea que implique cálculo o análisis de datos en el entorno administrativo.

5.2 La interfaz de Excel 365

6. LIBROS, HOJAS Y CELDAS

6.1 El libro de Excel

Un libro (workbook) es el archivo de Excel, que se guarda con extensión .xlsx (o .xlsm si contiene macros). Cada libro puede contener múltiples hojas de cálculo.

6.2 Las hojas de cálculo

Cada libro está formado por una o varias hojas de cálculo (worksheets), que aparecen como pestañas en la parte inferior de la pantalla. Por defecto, un libro nuevo tiene una sola hoja llamada «Hoja1».

Operaciones con hojas:

  • Insertar hoja: clic en el icono «+» junto a las pestañas, o clic derecho en una pestaña → Insertar.
  • Eliminar hoja: clic derecho en la pestaña → Eliminar (¡atención: irreversible!).
  • Cambiar nombre: doble clic en la pestaña o clic derecho → Cambiar nombre.
  • Mover o copiar hoja: clic derecho → Mover o copiar; permite reordenar hojas o copiarlas a otro libro.
  • Color de pestaña: clic derecho → Color de pestaña; para organizar visualmente las hojas.

6.3 Las celdas

La celda es la unidad básica de la hoja de cálculo: la intersección de una fila y una columna. Cada celda se identifica por su referencia, formada por la letra de la columna y el número de la fila (ej. B3, D12, AA5).

La celda que está actualmente seleccionada se denomina celda activa y se identifica por el borde resaltado y por su referencia en el cuadro de nombres.

6.4 Rangos

Un rango es un conjunto de celdas contiguas. Se identifica por la referencia de la celda superior izquierda y la inferior derecha, separadas por dos puntos (ej. A1:D10 es el rango que abarca desde la celda A1 hasta la D10).

Los rangos pueden:

  • Seleccionarse arrastrando el ratón o usando Mayúsculas + clic.
  • Recibir un nombre desde el cuadro de nombres o desde Fórmulas → Definir nombre, lo que facilita su uso en fórmulas (ej. =SUMA(VentasEnero) en lugar de =SUMA(B2:B31)).

7. CONFIGURACIÓN DE LA HOJA

7.1 Ancho de columnas y alto de filas

  • Ajustar ancho de columna: arrastrar el borde derecho del encabezado de la columna, o doble clic para autoajustar al contenido, o clic derecho en el encabezado → Ancho de columna.
  • Ajustar alto de fila: igual que las columnas pero con el borde inferior del encabezado de fila.
  • Ocultar/mostrar filas o columnas: clic derecho en el encabezado → Ocultar / Mostrar.

7.2 Inmovilizar paneles

La opción Vista → Inmovilizar paneles permite fijar filas y/o columnas para que permanezcan visibles al desplazarse por una hoja con muchos datos. Es muy útil cuando los encabezados de columna deben seguir visibles al bajar por una larga lista de datos.

7.3 Formato de celdas

El cuadro de diálogo de Formato de celdas (Ctrl + 1) contiene seis pestañas:

PestañaOpciones

AlineaciónAlineación horizontal y vertical del contenido, orientación del texto, ajuste de texto (wrapping), combinar celdas

7.4 Configurar página e impresión

Desde Diseño de página se configuran:

  • Márgenes, orientación y tamaño del papel.
  • Área de impresión: selección de las celdas que se van a imprimir.
  • Saltos de página: para controlar dónde comienza una nueva página al imprimir.
  • Repetir filas/columnas en cada página: para que los encabezados aparezcan en todas las páginas impresas.
  • Escala: ajuste del tamaño de la impresión para que quepa en un número determinado de páginas.

La vista previa de impresión se accede desde Archivo → Imprimir (Ctrl + P).

8. INTRODUCCIÓN Y EDICIÓN DE DATOS

8.1 Tipos de datos en Excel

Excel reconoce automáticamente los siguientes tipos de datos al introducirlos en una celda:

TipoDescripciónAlineación predeterminada
TextoCualquier cadena de caracteres que Excel no reconoce como número, fecha o fórmulaIzquierda
NúmeroValores numéricos (enteros, decimales)Derecha

8.2 Introducción de datos

  • Seleccionar la celda y escribir el dato; pulsar Enter (confirma y baja) o Tab (confirma y se mueve a la derecha) o Esc (cancela).
  • Para introducir un número como texto (ej. un código postal que empieza por 0), preceder el valor con un apóstrofo: ‘09001.
  • Para introducir texto en varias líneas dentro de una misma celda: Alt + Enter.

8.3 Edición de datos

  • Editar el contenido de una celda: doble clic en la celda o tecla F2.
  • Eliminar el contenido: tecla Supr (elimina el contenido pero no el formato).
  • Eliminar contenido y formato: clic derecho → Eliminar contenido o desde Inicio → Borrar → Borrar todo.

8.4 Autorelleno (relleno automático)

El autorelleno permite extender una serie de datos a partir de uno o varios valores iniciales, arrastrando el controlador de relleno (pequeño cuadrado negro en la esquina inferior derecha de la celda o rango seleccionado):

  • Series numéricas: si hay dos celdas con valores 1 y 2, Excel continúa con 3, 4, 5…
  • Fechas: continúa la serie de días, meses o años según el patrón.
  • Texto fijo: copia el mismo texto en las celdas siguientes.
  • Listas predefinidas: Excel reconoce series como los días de la semana (Lunes, Martes…) y los meses (Enero, Febrero…).

8.5 Copiar, cortar y pegar en Excel

El pegado especial (Ctrl + Alt + V) es una función muy útil en Excel que permite elegir exactamente qué se pega:

Opción de pegado especialDescripción
TodoCopia el contenido y el formato (equivale a Ctrl+V normal)
FórmulasPega solo las fórmulas, sin el formato
ValoresPega el resultado de las fórmulas como valores fijos (elimina la fórmula)
FormatosPega solo el formato, sin el contenido
TransponerGira la selección: las filas pasan a ser columnas y viceversa

9. FÓRMULAS Y FUNCIONES

9.1 Fórmulas

Una fórmula es una expresión que calcula un valor a partir de operaciones matemáticas, referencias a celdas, funciones o una combinación de estos. Toda fórmula en Excel comienza con el signo =.

Operadores aritméticos en Excel:

OperadorOperaciónEjemplo
+Suma=A1+B1
-Resta=A1-B1
*Multiplicación=A1*B1
/División=A1/B1
^Potencia=A1^2
%Porcentaje=A1*15%

Operadores de comparación (devuelven VERDADERO o FALSO):

OperadorSignificado
=Igual
<>Distinto
>Mayor que
<Menor que
>=Mayor o igual
<=Menor o igual

9.2 Referencias a celdas

TipoEjemploDescripción

10. GRÁFICOS EN EXCEL

10.1 Creación de gráficos

Los gráficos permiten representar visualmente los datos de la hoja de cálculo. Para crear un gráfico:

  1. Seleccionar el rango de datos que se quiere representar (incluidos los encabezados).
  2. Ir a Insertar → grupo Gráficos y elegir el tipo de gráfico.
  3. El gráfico se inserta en la hoja como objeto flotante. Puede moverse a una hoja de gráfico independiente desde Diseño de gráfico → Mover gráfico.

10.2 Tipos de gráficos principales

Una vez creado el gráfico, aparecen las fichas contextuales Diseño de gráfico y Formato, desde las que se puede:

  • Cambiar el tipo de gráfico.
  • Seleccionar un diseño rápido (combinación de elementos predefinida).
  • Cambiar los colores del gráfico.
  • Añadir o modificar elementos del gráfico: título, leyenda, etiquetas de datos, eje, líneas de cuadrícula, tabla de datos.
  • Aplicar estilos de gráfico predefinidos.
  • Cambiar los datos de origen del gráfico.

11. GESTIÓN DE DATOS EN EXCEL

11.1 Ordenar datos

Desde Datos → Ordenar o los botones rápidos A→Z y Z→A, se pueden ordenar los datos de una lista por una o varias columnas, en orden ascendente o descendente. En el cuadro de diálogo de ordenación se pueden añadir múltiples niveles de ordenación (primero por columna A, luego por columna B, etc.).

Importante: si los datos tienen encabezados de columna, activar la opción Mis datos tienen encabezados para que los encabezados no se incluyan en la ordenación.

11.2 Filtrar datos (Autofiltro)

Los filtros permiten mostrar solo las filas que cumplen determinados criterios, ocultando el resto temporalmente.

Para aplicar autofiltro:

  1. Seleccionar cualquier celda de la lista de datos.
  2. Datos → Filtro (o Ctrl + Mayúsc + L).
  3. Aparecen flechas desplegables en los encabezados de columna.
  4. Hacer clic en la flecha de la columna que se quiere filtrar y elegir los criterios.

Los filtros pueden ser:

  • Por valor: seleccionar qué valores mostrar.
  • Filtros de número: mayor que, menor que, entre, los 10 primeros, por encima del promedio…
  • Filtros de texto: comienza por, contiene, termina en…
  • Filtros de fecha: hoy, esta semana, este mes, entre fechas…

Para quitar el filtro: Datos → Filtro de nuevo, o clic en la flecha de la columna → Borrar filtro.

11.3 Tablas de Excel

Convertir un rango de datos en una Tabla de Excel (Ctrl + T o Insertar → Tabla) ofrece ventajas adicionales:

  • Las fórmulas se extienden automáticamente al añadir filas.
  • El autofiltro se activa automáticamente.
  • Se pueden aplicar estilos de tabla predefinidos.
  • Las tablas tienen nombre propio y sus columnas pueden referenciarse por nombre en fórmulas.

11.4 Tablas dinámicas (PivotTable)

Las tablas dinámicas son una de las funciones más potentes de Excel. Permiten resumir, analizar, explorar y presentar datos de una lista o tabla de forma dinámica e interactiva, sin modificar los datos originales.

Para crear una tabla dinámica:

  1. Seleccionar cualquier celda de los datos de origen.
  2. Insertar → Tabla dinámica.
  3. En el panel de campos, arrastrar los campos a las zonas: Filtros, Columnas, Filas y Valores.

Con una tabla dinámica se puede, por ejemplo, mostrar el total de ventas por mes y por empleado, o el número de expedientes por tipo y por departamento, reorganizando el análisis simplemente arrastrando campos.

11.5 Formato condicional

El formato condicional (Inicio → Formato condicional) aplica automáticamente un formato (color de fondo, color de fuente, iconos, barras de datos) a las celdas que cumplen determinados criterios, facilitando la interpretación visual de los datos.

Ejemplos:

  • Resaltar en rojo las celdas con valores negativos.
  • Aplicar una escala de colores de verde a rojo según el valor.
  • Mostrar barras de datos proporcionales al valor.
  • Resaltar los valores duplicados.

11.6 Validación de datos

La validación de datos (Datos → Validación de datos) permite restringir qué tipo de datos puede introducirse en una celda o rango, evitando errores:

  • Solo números enteros entre 1 y 100.
  • Solo fechas entre dos fechas concretas.
  • Solo valores de una lista desplegable predefinida.
  • Solo texto de longitud máxima determinada.

Cuándo el usuario intenta introducir un dato inválido, Excel muestra un mensaje de error configurable.
