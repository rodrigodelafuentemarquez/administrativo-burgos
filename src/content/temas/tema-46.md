│


├── FUNDAMENTO Y ÁMBITO


│   ├── Art. 149.1.18 CE: Estado → bases (TREBEP) / CyL → desarrollo (Ley 7/2005)


│   └── Aplica a: funcionarios carrera + interinos + eventual de la Junta CyL


│


├── CLASES DE PERSONAL


│   ├── Funcionario de carrera: vinculación permanente / adquiere condición con nombramiento + posesión


│   │   Pérdida: renuncia / pérdida nacionalidad / jubilación / separación / inhabilitación


│   ├── Funcionario interino: necesidad urgente / plaza vacante / sustitución / programa temporal /


│   │   exceso tareas (max. 6 meses en 12) / cese cuando desaparece la causa


│   ├── Personal eventual: confianza y asesoramiento / libre nombramiento y cese /


│   │   cesa automáticamente con el cargo político / no ejerce potestades administrativas


│   └── Personal directivo: méritos + capacidad + idoneidad / evaluación periódica


│


├── CUERPOS Y ESCALAS EN JUNTA CyL


│   ├── A1: Cuerpo Superior / A2: Cuerpo de Gestión


│   ├── C1: Cuerpo Administrativo → las oposiciones que preparamos


│   ├── C2: Cuerpo Auxiliar / Agrupaciones profesionales


│   └── Cuerpo Administrativo C1: funciones de gestión administrativa, tramitación de expedientes,


│       atención al ciudadano, registro, archivo, gestión económica básica


│


├── ACCESO AL EMPLEO PÚBLICO


│   ├── Principios: igualdad / mérito / capacidad / publicidad / transparencia / imparcialidad


│   ├── OEP: aprueba Consejo de Gobierno / BOCYL + BOE / convocatoria en 3 años max.


│   ├── Requisitos: nacionalidad / capacidad funcional / 16 años-jubilación / no separación /


│   │   titulación del subgrupo / pago derechos de examen


│   └── Sistemas selectivos:


│       ├── Oposición: solo pruebas → sistema ordinario / mejor garantía mérito y capacidad


│       ├── Concurso-oposición: oposición (eliminatoria) + valoración méritos → sistema mixto


│       └── Concurso: solo méritos → excepcional / personal laboral / estabilización


│


├── PROVISIÓN DE PUESTOS (RPT: instrumento de ordenación)


│   ├── Concurso de méritos: sistema ORDINARIO / grado personal + experiencia + formación


│   │   → genera reserva del puesto


│   ├── Libre designación: puestos de especial responsabilidad o confianza / cese libre


│   ├── Adscripción provisional: carácter temporal / obligación de concursar


│   └── Comisión de servicios: temporal / voluntaria o forzosa / conserva puesto de origen


│


├── CARRERA PROFESIONAL


│   ├── Carrera vertical: ascenso a puestos superiores via concursos de méritos


│   ├── Grado personal: nivel de C. Destino consolidado / 2 años consecutivos o 3 con interrupción


│   │   → derecho a percibir mínimo el C. Destino del grado aunque ocupe puesto inferior


│   ├── Carrera horizontal: progresión sin cambiar puesto / evaluación del desempeño


│   └── Promoción interna: ascenso de subgrupo / proceso selectivo específico + titulación


│


├── SITUACIONES ADMINISTRATIVAS


│   ├── Servicio activo: situación ordinaria / todas las retribuciones


│   ├── Servicios especiales: cargo público incompatible / reserva puesto + cómputo trienios


│   ├── Servicio en otras AAPP: prestación en otra Administración


│   ├── Excedencia voluntaria (interés particular): min. 2 años / sin retribución / sin reserva


│   ├── Excedencia cuidado hijos: max. 3 años / 1er año = servicio activo + reserva puesto


│   ├── Excedencia por violencia de género: hasta 18 meses como servicio activo


│   └── Suspensión de funciones: provisional (expediente/proceso penal) o firme (sanción)


│


├── DERECHOS Y DEBERES


│   ├── Derechos: inamovilidad / retribuciones / carrera / formación / no discriminación /


│   │   conciliación / sindicación / huelga / negociación colectiva


│   └── Deberes: diligencia / objetividad / obediencia (salvo infracción manifiesta) /


│       secreto profesional / respeto a ciudadanos y compañeros / neutralidad política


│


├── RÉGIMEN DISCIPLINARIO


│   ├── Principios: legalidad / tipicidad / culpabilidad / proporcionalidad /


│   │   non bis in idem / presunción de inocencia


│   ├── Infracciones muy graves: separación / suspensión 3-6 años / traslado forzoso


│   │   Prescripción: 3 años / sanción: 3 años


│   ├── Infracciones graves: suspensión 1 día-3 años / traslado / demérito


│   │   Prescripción: 2 años / sanción: 2 años


│   ├── Infracciones leves: apercibimiento / deducción retribuciones


│   │   Prescripción: 6 meses / sanción: 1 año


│   └── Procedimiento: expediente disciplinario contradictorio (salvo leves: sumario)


│


└── NEGOCIACIÓN COLECTIVA


    ├── Mesa General de Negociación de Empleados Públicos CyL: paritaria AAPP-sindicatos


    ├── Materias: retribuciones / OEP / formación / carrera / concursos / conciliación


    └── Huelga: derecho constitucional (art. 28.2 CE) / con mantenimiento de servicios mínimos


        → incumplimiento servicios mínimos = posible responsabilidad disciplinaria
