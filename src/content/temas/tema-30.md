├── PARTE I: SEGURIDAD SOCIAL


│   │


│   ├── ENCUADRAMIENTO


│   │   └── Todo el personal local (funcionario y laboral) → RÉGIMEN GENERAL


│   │       (MUNPAL suprimida e integrada en el RG desde 1993)


│   │   → Corporación Local = empresario cotizante (inscripción + afiliación + cotización)


│   │


│   ├── ACCIÓN PROTECTORA (art. 42 TRLGSS)


│   │   └── Asistencia sanitaria / IT / Nacimiento y cuidado de menor /


│   │       Riesgo embarazo / IP / Jubilación / Muerte y supervivencia /


│   │       Desempleo / Prestaciones familiares


│   │


│   ├── INCAPACIDAD TEMPORAL (IT)


│   │   ├── Contingencias: enf. común / acc. no laboral / acc. trabajo / enf. profesional


│   │   ├── Subsidio: días 1-3 sin subsidio / días 4-20: 60% BR / día 21+: 75% BR


│   │   │   → Contingencias profesionales: 75% desde el día siguiente a la baja


│   │   │   → Funcionarios: complemento corporativo hasta el 100%


│   │   └── Duración: máx. 365 días + prórr. 180 días = 545 días máx.


│   │


│   ├── NACIMIENTO Y CUIDADO DE MENOR (antes maternidad/paternidad)


│   │   ├── 16 semanas para cada progenitor (6 obligatorias e ininterrumpidas)


│   │   ├── Cuantía: 100% de la base reguladora


│   │   └── Requisito: 180 días cotizados en últimos 7 años (o 360 en toda la vida)


│   │


│   └── INCAPACIDAD PERMANENTE


│       ├── IPP: < 33% capacidad → indemnización a tanto alzado (24 mensualidades BR)


│       ├── IPT: inhabilita para profesión habitual → pensión 55% BR (75% si > 55 años: IPT cualificada)


│       ├── IPA: inhabilita para toda profesión → pensión 100% BR


│       └── Gran invalidez: necesita ayuda de 3ª persona → 100% BR + complemento asistencial


│


└── PARTE II: PRESUPUESTO LOCAL


    │


    ├── CONCEPTO: expresión cifrada de obligaciones e ingresos previstos para el ejercicio


    │   → Carácter anual (año natural) / Principio de equilibrio presupuestario


    │


    ├── ESTRUCTURA


    │   ├── INGRESOS (Cap. 1-9): corrientes (1-5) / capital (6-7) / financieros (8-9)


    │   └── GASTOS (Cap. 1-9): corrientes (1-4) / capital (6-7) / financieros (8-9)


    │       → Triple clasificación del gasto: orgánica (quién) / funcional (para qué) / económica (en qué)


    │


    ├── APROBACIÓN


    │   └── Elaboración → Informe Intervención → Aprobación inicial Pleno →


    │       Exposición pública 15 días → Aprobación definitiva → Publicación BOP


    │       → Si no se aprueba antes del 1/1: PRÓRROGA automática del anterior


    │


    ├── CICLO DEL GASTO Y DOCUMENTOS CONTABLES


    │   ├── A  → Autorización (crédito retenido)


    │   ├── D  → Disposición/Compromiso (crédito comprometido)


    │   ├── O  → Reconocimiento de obligación (crédito obligado)


    │   ├── P  → Ordenación del pago (orden a Tesorería)


    │   └── Pago material (salida efectiva de fondos)


    │   → Documentos combinados frecuentes: AD / ADO / ADOP


    │


    ├── ÓRGANOS COMPETENTES


    │   ├── Pleno: aprobar Presupuesto + Cuenta General + tributos + endeudamiento mayor


    │   ├── Presidente: ordenar pagos + autorizar gastos (dentro de sus límites)


    │   ├── Intervención: fiscalización previa + contabilidad + control interno


    │   └── Tesorería: recaudación + custodia de fondos + pago de obligaciones


    │


    └── CUENTA GENERAL


        └── Intervención → Comisión Especial de Cuentas → Exposición pública 15 días


            → Aprobación por Pleno → Remisión al Tribunal de Cuentas
