│


├── ORIGEN Y EVOLUCIÓN DE INTERNET


│   ├── 1969: ARPANET — primera red con conmutación de paquetes (Dpto. Defensa EE.UU.)


│   │         Primera conexión: UCLA ↔ Stanford Research Institute (29/10/1969)


│   ├── 1974-1983: TCP/IP — Vinton Cerf y Robert Kahn / estándar de ARPANET en 1983


│   ├── 1991: World Wide Web — Tim Berners-Lee (CERN) / hipertexto + HTTP + navegador


│   │         Primera página web: agosto 1991 / Primer navegador visual: Mosaic (1993)


│   ├── 1990s: buscadores (Yahoo!, AltaVista, Google 1998) / e-commerce / burbuja .com (2000)


│   ├── 2004+: Web 2.0 — participación del usuario / blogs / wikis / redes sociales


│   │           (Facebook 2004, YouTube 2005, Twitter 2006)


│   ├── 2007+: Web móvil (iPhone) → más tráfico desde móvil que desde PC / IoT


│   └── Actualidad: +5.000 M usuarios / fibra / 5G / IA / cloud / regulación UE (RGPD, DSA, DMA)


│


├── PROTOCOLOS Y SERVICIOS


│   ├── TCP/IP: protocolo fundamental de internet


│   │   ├── IP: direccionamiento y enrutamiento de paquetes


│   │   │   → IPv4: 32 bits / cuatro grupos decimales (ej. 192.168.1.1) / ~4.000 M dir.


│   │   │   → IPv6: 128 bits / ocho grupos hexadecimales / prácticamente ilimitado


│   │   ├── TCP: transmisión FIABLE (garantiza entrega, orden y retransmisión)


│   │   └── UDP: transmisión rápida sin garantía de entrega (streaming, videollamadas)


│   ├── DNS: traduce nombres de dominio en direcciones IP


│   │   → Estructura: protocolo://subdominio.dominio.TLD/ruta


│   │   → TLD: .com / .org / .es / .gob.es / .eu / .edu


│   ├── HTTP / HTTPS


│   │   ├── HTTP: protocolo de transferencia de hipertexto (petición-respuesta cliente-servidor)


│   │   │   → Códigos: 200 OK / 301-302 redirección / 404 no encontrado / 500 error servidor


│   │   └── HTTPS: HTTP + cifrado SSL/TLS → confidencialidad + integridad + autenticidad


│   │       → Candado en el navegador / obligatorio para sedes electrónicas y datos personales


│   ├── Correo electrónico:


│   │   ├── SMTP: envío de correos (cliente → servidor y entre servidores)


│   │   ├── POP3: descarga mensajes del servidor (los elimina del servidor por defecto)


│   │   └── IMAP: accede a mensajes en el servidor sin descargarlos (sincronización multi-dispositivo)


│   │       → IMAP ha desplazado a POP3 por permitir acceso desde varios dispositivos


│   ├── FTP / SFTP: transferencia de archivos entre sistemas


│   ├── VPN: túnel cifrado a través de internet → acceso remoto seguro (teletrabajo)


│   ├── URL: protocolo + dominio + ruta (+ parámetros + ancla)


│   └── Cloud Computing: IaaS / PaaS / SaaS


│       → SaaS relevante: Microsoft 365, Google Workspace (acceso vía navegador)


│


└── NAVEGADORES WEB


    ├── Concepto: programa que interpreta HTML/CSS/JS y muestra páginas web


    ├── Principales: Chrome (Blink, ~65%) / Edge (Blink, pred. Windows 11) /


    │               Firefox (Gecko) / Safari (WebKit, Apple)


    ├── Interfaz: barra de dirección (omnibox) / botones ←→↺ / pestañas /


    │            barra de favoritos / candado de seguridad / menú principal


    ├── FUNCIONALIDADES


    │   ├── Navegación: escribir URL / buscar desde omnibox / Atrás-Adelante / F5 (actualizar) /


    │   │              Ctrl+F5 (actualización forzada sin caché) / Esc (detener)


    │   ├── Pestañas: Ctrl+T (nueva) / Ctrl+W (cerrar) / Ctrl+Mayúsc+T (reabrir cerrada)


    │   │             Ctrl+Mayúsc+N (incógnito/InPrivate)


    │   ├── Favoritos/Marcadores: Ctrl+D (añadir) / organizar en carpetas / importar-exportar


    │   ├── Historial: Ctrl+H / ver y eliminar páginas visitadas


    │   ├── Descargas: Ctrl+J / gestionar archivos descargados


    │   ├── Búsqueda en página: Ctrl+F → resalta coincidencias


    │   ├── Zoom: Ctrl+ / Ctrl- / Ctrl+0 (restaurar 100%)


    │   └── Pantalla completa: F11


    ├── PRIVACIDAD Y SEGURIDAD


    │   ├── Borrar datos (Ctrl+Mayúsc+Supr): historial / cookies / caché / contraseñas


    │   ├── Navegación privada/incógnito: no guarda historial ni cookies al cerrar


    │   │   ¡ATENCIÓN! No hace anónimo al usuario frente al ISP, empleador ni sitios web


    │   ├── Cookies: sesión / persistentes / propias / terceros (publicidad)


    │   │   → RGPD: banner de cookies + consentimiento para no esenciales


    │   └── Extensiones: añaden funcionalidades / solo instalar de fuentes confiables


    └── En el entorno administrativo:


        → Acceso a sedes electrónicas / expedientes / Portal Transparencia /


          Microsoft 365 web / DEHú / Carpeta Ciudadana


        → AutoFirma: aplicación para firmar con certificado electrónico desde el navegador
