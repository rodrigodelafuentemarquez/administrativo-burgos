│


├── HARDWARE


│   ├── CPU (procesador): UC (coordina) + UAL (opera) + registros + caché L1/L2/L3


│   │   → Velocidad en GHz / núcleos (dual, quad, octa-core) / fabricantes: Intel y AMD


│   ├── RAM: memoria principal, VOLÁTIL, se pierde al apagar / capacidad en GB / DDR4-DDR5


│   ├── ROM / BIOS-UEFI: memoria no volátil / firmware de arranque / POST


│   ├── Placa base: conecta todos los componentes / chipset / zócalo CPU / ranuras RAM / puertos


│   ├── SSD vs HDD:


│   │   ├── HDD: magnético, partes móviles, gran capacidad a bajo coste, más lento y frágil


│   │   └── SSD: flash, sin partes móviles, mucho más rápido, resistente, mayor precio por GB


│   │       → Interfaces SSD: SATA (~550 MB/s) / M.2 NVMe PCIe (>3.000 MB/s)


│   └── Periféricos: entrada (teclado, ratón, escáner) / salida (monitor, impresora) / mixtos


│


├── SOFTWARE


│   ├── De sistema: SO + drivers + BIOS/UEFI + utilidades


│   ├── De aplicación: Word, Excel, navegadores, correo, ERP…


│   ├── De programación: compiladores, IDEs, depuradores


│   └── Por licencia: propietario / libre (código abierto) / freeware / shareware / SaaS


│


├── SISTEMAS OPERATIVOS


│   ├── Funciones: gestión de CPU + RAM + almacenamiento + periféricos + usuarios + red + interfaz


│   ├── Tipos de interfaz: CLI (línea de comandos) vs. GUI (gráfica)


│   ├── Principales SO:


│   │   ├── Windows 11 (Microsoft): dominante en escritorio y en AAPP españolas


│   │   ├── Linux (código abierto): dominante en servidores / múltiples distribuciones


│   │   ├── macOS (Apple): uso minoritario en entornos de trabajo administrativo


│   │   └── Android / iOS: sistemas móviles


│   ├── Conceptos clave: proceso / multitarea / memoria virtual /


│   │   sistema de archivos (NTFS en Windows, ext4 en Linux) / usuarios y permisos


│   └── Administrador vs. usuario estándar: el administrador tiene control total del sistema


│


├── SISTEMAS DE ALMACENAMIENTO


│   ├── HDD: magnético / 5.400-7.200 RPM / 1 TB-varios TB / lento / barato / frágil


│   ├── SSD: flash / sin partes móviles / muy rápido / robusto / más caro por GB


│   ├── Ópticos: CD (700 MB) / DVD (4,7-8,5 GB) / Blu-ray (25-50 GB) / en declive


│   ├── Flash portátil: USB/pendrive + tarjetas SD/microSD


│   ├── Almacenamiento en red: NAS (oficinas) / SAN (grandes empresas)


│   ├── Nube: OneDrive / Google Drive / iCloud / Dropbox


│   └── Unidades: bit → byte (8b) → KB → MB → GB → TB → PB


│


└── SEGURIDAD INFORMÁTICA


    ├── Triada CIA: Confidencialidad + Integridad + Disponibilidad


    ├── Marco normativo AAPP: Esquema Nacional de Seguridad — ENS (RD 3/2010, actualiz. RD 311/2022)


    │   → Categorías: básica / media / alta según impacto del fallo


    ├── Amenazas:


    │   ├── Malware: virus / gusano / troyano / RANSOMWARE (cifra y pide rescate) /


    │   │           spyware / keylogger / rootkit


    │   ├── Phishing / ingeniería social: suplantación de identidad para robar credenciales


    │   ├── Ataques a contraseñas: fuerza bruta / diccionario / credential stuffing


    │   └── DoS/DDoS: saturación del servicio para hacerlo inaccesible


    ├── Medidas de protección:


    │   ├── Antivirus / antimalware → actualizado (Windows: Defender integrado)


    │   ├── Firewall (cortafuegos): controla tráfico de red


    │   ├── Actualizaciones del SO y aplicaciones: corrigen vulnerabilidades


    │   ├── Contraseñas seguras: larga + mixta (may/min/núm/especiales) + única por servicio


    │   ├── 2FA (doble factor): contraseña + OTP/biometría/token físico


    │   ├── Copias de seguridad (backup): regla 3-2-1 (3 copias / 2 soportes / 1 offsite)


    │   ├── Cifrado: BitLocker (disco) / HTTPS (navegación) / VPN (red)


    │   └── Buenas prácticas: no abrir adjuntos sospechosos / bloquear equipo / no conectar USB


    │                          desconocidos / no usar Wi-Fi pública sin VPN / reportar incidencias


    └── ENS: principios: seguridad integral / gestión de riesgos / prevención-detección-respuesta-recuperación

     │                          / mejora continua
