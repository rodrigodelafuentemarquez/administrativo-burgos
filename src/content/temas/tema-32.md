│


├── CONCEPTO DE DOCUMENTO ADMINISTRATIVO (art. 26 LPACAP)


│   ├── Producido o recibido por una AAPP en el ejercicio de sus funciones


│   ├── Características: autenticidad + integridad + fiabilidad + disponibilidad + eficacia probatoria


│   ├── Documento electrónico: requiere firma electrónica / sello electrónico / CSV


│   └── Copias: auténtica (misma validez que original) / compulsada / simple (informativa)


│


├── CLASIFICACIÓN DE DOCUMENTOS ADMINISTRATIVOS


│   ├── Por naturaleza del acto:


│   │   ├── Decisión: resolución / acuerdo / decreto / orden de servicio


│   │   ├── Constancia: acta / diligencia / certificado


│   │   ├── Transmisión: oficio / nota interior / carta / circular / notificación


│   │   ├── Juicio: informe (preceptivo/facultativo; vinculante/no vinculante) /


│   │   │          propuesta de resolución / dictamen


│   │   └── De los ciudadanos: solicitud / alegaciones / recurso / denuncia


│   ├── Por soporte: papel / electrónico / multimedia / audiovisual


│   ├── Por confidencialidad: públicos / restringidos / confidenciales


│   └── Por fase del procedimiento: inicio / instrucción / terminación / ejecución


│


├── ARCHIVO DE DOCUMENTOS ADMINISTRATIVOS


│   ├── Funciones: administrativa + jurídica + histórica + informativa


│   ├── CICLO VITAL DEL DOCUMENTO:


│   │   ├── Archivo de gestión (1-5 años): expedientes activos, consulta frecuente


│   │   ├── Archivo central/intermedio (5-25 años): documentos semiactivos


│   │   └── Archivo histórico (indefinido): valor histórico → Patrimonio Documental


│   ├── Organización: cuadro de clasificación + inventario + catálogo + guía + índice


│   ├── Ordenación: cronológica / alfabética / numérica / geográfica


│   ├── Evaluación documental: conservación permanente vs. eliminación (expediente de expurgo)


│   │   → Eliminar sin autorización = infracción (Ley 16/1985)


│   └── Gestión electrónica: ENI (Real Decreto 4/2010) / expediente electrónico (art. 70 LPACAP)


│       → Conservación, migración de formatos, trazabilidad, seguridad


│


├── DERECHO DE ACCESO (doble régimen)


│   ├── Acceso de interesados en procedimiento (art. 53 LPACAP):


│   │   → Derecho de interesados: estado de tramitación + copia del expediente


│   │   → No requiere motivación especial; basta con ser interesado


│   └── Derecho de acceso a información pública (LTAIBG Ley 19/2013):


│       → Cualquier persona, sin necesidad de ser interesado ni motivar


│       → Plazo de resolución: 1 MES (prorrogable 1 mes más)


│       → Silencio: NEGATIVO (desestimación)


│       → Denegación deben motivarse y puede reclamarse ante Consejo de Transparencia


│


├── LÍMITES AL ACCESO (art. 14 LTAIBG — interpretación RESTRICTIVA)


│   ├── Seguridad nacional / defensa / relaciones exteriores / seguridad pública


│   ├── Prevención e investigación de infracciones penales o administrativas


│   ├── Igualdad de partes en procesos judiciales


│   ├── Funciones de inspección y control


│   ├── Intereses económicos y comerciales / secreto profesional / propiedad intelectual


│   ├── Confidencialidad del proceso de toma de decisiones


│   └── Protección de datos personales (límite más frecuente):


│       ├── Datos especialmente protegidos (ideología, salud, origen racial…): DENEGACIÓN salvo consentimiento


│       └── Otros datos: ponderación → si prevalece acceso, se puede ANONIMIZAR


│


└── FORMAS DE ACCESO


    ├── Modalidades: consulta in situ / copia papel / copia electrónica /


    │               comunicación verbal / acceso remoto a BD / transposición de formato


    ├── TRANSPARENCIA ACTIVA (publicidad proactiva):


    │   → Portal de Transparencia


    │   → Información institucional + jurídica + económico-presupuestaria


    │   → Actualizada / accesible / comprensible / reutilizable


    └── TRANSPARENCIA PASIVA (acceso a solicitud):


        → Solicitud → 1 mes para resolver → estimación / denegación motivada


        → Reclamación ante Consejo de Transparencia (potestativa, previa al contencioso)
