│


├── FUNDAMENTOS


│   ├── Windows 11 (oct. 2021): requisitos TPM 2.0 + Secure Boot + CPU 64 bits


│   ├── Ediciones: Home / Pro (uso admin.) / Enterprise (grandes orgs.) / Education


│   └── Novedades: Inicio centrado / Snap Layouts / Copilot / mayor seguridad


│


├── ESCRITORIO Y BARRA DE TAREAS


│   ├── Escritorio: área de trabajo principal con iconos y ventanas


│   ├── Barra de tareas (inferior): Inicio + iconos anclados + apps abiertas +


│   │   área de notificación + búsqueda + Vista de tareas + Widgets


│   └── Área de notificación: reloj + red + volumen + batería + Centro de actividades


│


├── MENÚ INICIO (⊞ o tecla Windows)


│   ├── Búsqueda + Apps ancladas + Recomendados (recientes) + Todas las aplicaciones


│   └── Opciones de encendido: Suspender / Apagar / Reiniciar


│


├── COPILOT (⊞ + C)


│   ├── IA integrada en Windows 11 (desde fin 2023)


│   └── Funciones: preguntas / búsqueda web / acciones del sistema / generar texto /


│               resumir documentos / asistencia en apps Microsoft 365


│


├── VENTANAS


│   ├── Elementos: barra de título + barra de menús + área de contenido +


│   │             barras de desplazamiento + botones Minimizar(-) / Maximizar(□) / Cerrar(✕)


│   ├── Operaciones: mover / redimensionar / minimizar / maximizar / restaurar / cerrar


│   ├── Alt + Tab: cambiar entre ventanas


│   └── Snap Layouts (⊞ + Z): organización de ventanas en disposiciones predefinidas


│


├── MENÚS CONTEXTUALES Y CUADROS DE DIÁLOGO


│   ├── Menú contextual: clic derecho → acciones del elemento seleccionado


│   │   → En Windows 11: menú simplificado + «Mostrar más opciones» para el menú completo


│   └── Cuadros de diálogo: confirmación / alerta / configuración / entrada de datos


│       Elementos: botones / checkboxes / botones de radio / listas desplegables / pestañas


│


├── EXPLORADOR DE ARCHIVOS (⊞ + E)


│   ├── Panel de navegación (izq.): Acceso rápido / OneDrive / Este equipo / Red


│   ├── Área de contenido (dcha.): archivos y carpetas de la ubicación actual


│   ├── Barra de dirección: ruta actual / navegación por clic o escritura


│   └── Vistas: iconos / lista / DETALLES (más informativa) / mosaicos / contenido


│


├── GESTIÓN DE ARCHIVOS Y CARPETAS


│   ├── Extensiones clave: .docx / .xlsx / .pptx / .pdf / .txt / .jpg / .zip / .exe


│   ├── Crear: clic derecho → Nueva carpeta (Ctrl+Mayúsc+N) / Nuevo archivo


│   ├── Selección: clic / Mayúsc+clic (contiguos) / Ctrl+clic (no contiguos) / Ctrl+A (todos)


│   ├── Copiar (Ctrl+C) / Cortar (Ctrl+X) / Pegar (Ctrl+V)


│   │   → Arrastrar mismo disco = MOVER / arrastrar disco distinto = COPIAR


│   ├── Cambiar nombre: F2


│   ├── Eliminar: Supr → Papelera / Mayúsc+Supr → eliminación definitiva (¡irreversible!)


│   └── Papelera: almacena eliminados del disco interno → Restaurar o Vaciar


│


├── BÚSQUEDA


│   ├── Desde el Explorador: campo superior derecho → busca en carpeta actual y subcarpetas


│   └── Desde barra de tareas: apps + archivos + configuración + web (Bing)


│       → Indexación de búsqueda: en segundo plano / configurable en Panel de control


│


├── ESTE EQUIPO Y ACCESO RÁPIDO


│   ├── Este equipo: todas las unidades de almacenamiento + capacidad y espacio libre


│   └── Acceso rápido: carpetas ancladas + archivos recientes


│       → Anclar: clic derecho → «Anclar a Acceso rápido»


│


├── ACCESORIOS


│   ├── Bloc de notas: editor de texto sin formato (.txt) / pestañas en Win 11


│   ├── Paint: edición de imágenes básica / eliminación de fondo en Win 11


│   ├── Calculadora: estándar / científica / programador / conversores


│   ├── Recorte y anotación (⊞+Mayúsc+S): captura de pantalla parcial o total


│   └── Windows Media Player: reproducción de audio y vídeo


│


└── HERRAMIENTAS DEL SISTEMA


    ├── Administrador de tareas (Ctrl+Mayúsc+Esc): procesos + rendimiento + inicio + usuarios


    │   → Finalizar procesos bloqueados / desactivar apps de inicio lentas


    ├── Configuración (⊞+I): herramienta moderna de ajustes del sistema


    ├── Gestión de discos: particiones + formatear + asignar letras de unidad


    ├── Desfragmentador: HDD → desfragmenta / SSD → TRIM automático


    ├── Limpieza de disco: elimina archivos temporales e innecesarios


    ├── Comprobación de errores del disco (chkdsk): detecta y repara errores


    ├── Restauración del sistema: vuelve al equipo a un punto de restauración anterior


    ├── Windows Update (Configuración → Windows Update): actualizaciones de seguridad y sistema


    ├── Registro del sistema (regedit): base de datos de configuración → solo usuarios avanzados


    └── Visor de eventos: registro de errores, advertencias e información del sistema → diagnóstico
