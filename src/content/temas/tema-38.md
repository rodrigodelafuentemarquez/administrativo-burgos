│


├── WORD 365 — PROCESADOR DE TEXTO


│   ├── Interfaz: cinta de opciones (Inicio/Insertar/Diseño/Disposición/Referencias/


│   │            Correspondencia/Revisar/Vista) + barra de acceso rápido + regla


│   ├── FORMATO


│   │   ├── Fuente: tipo/tamaño/negrita(Ctrl+N)/cursiva(Ctrl+K)/subrayado(Ctrl+S)/color


│   │   └── Párrafo: alineación (izq/cent/dcha/justif) / interlineado / sangría /


│   │               listas viñetas y numeradas


│   ├── ESTILOS: Título 1/2/3 (generan tabla de contenido) + Normal + Énfasis


│   │   → Clave para estructura coherente y tabla de contenido automática


│   ├── ELEMENTOS DEL DOCUMENTO


│   │   ├── Encabezado/Pie: Insertar → Encabezado/Pie de página


│   │   ├── Número de página: Insertar → Número de página


│   │   ├── Tabla de contenido: Referencias → Tabla de contenido (basada en Títulos)


│   │   ├── Notas al pie/final: Referencias → Insertar nota al pie (Ctrl+Alt+F)


│   │   ├── Tablas: Insertar → Tabla


│   │   └── Combinar correspondencia: Correspondencia → doc. base + fuente de datos → campos


│   ├── GESTIÓN DE FICHEROS


│   │   ├── Guardar (Ctrl+G) / Guardar como (F12) / Autoguardado (en OneDrive)


│   │   ├── Formatos: .docx (pred.) / .pdf (exportar) / .rtf / .odt / .txt


│   │   └── Recuperación: Autorrecuperación configurable + historial de versiones (OneDrive)


│   ├── IMPRESIÓN (Ctrl+P): impresora / páginas / copias / orientación / páginas por hoja


│   ├── REVISIÓN: ortografía (F7) / seguimiento de cambios (Ctrl+Mayúsc+E) / comentarios


│   └── VISTAS: Diseño de impresión (pred.) / Lectura / Web / Esquema / Borrador


│       Panel de navegación (Ctrl+F): árbol de títulos + búsqueda de texto


│


└── EXCEL 365 — HOJA DE CÁLCULO


    ├── Interfaz: cuadro de nombres + barra de fórmulas + cuadrícula + pestañas de hoja


    ├── ESTRUCTURA: Libro (.xlsx) → Hojas → Celdas (referencia col+fila: A1, B3…)


    │   → Rango: conjunto de celdas contiguas (A1:D10)


    ├── TIPOS DE DATOS: texto (izq.) / número (dcha.) / fecha / fórmula (=) / lógico / error


    ├── FORMATO DE CELDAS (Ctrl+1): número / alineación / fuente / bordes / relleno / proteger


    │   → Formato de número: General / Número / Moneda / Fecha / Porcentaje / Personalizado


    ├── FÓRMULAS Y FUNCIONES


    │   ├── Toda fórmula empieza por =


    │   ├── Referencias: relativa (A1) / absoluta ($A$1, con F4) / mixta ($A1 o A$1)


    │   ├── Funciones matemáticas: SUMA / PROMEDIO / MAX / MIN / CONTAR / CONTAR.SI /


    │   │                          SUMAR.SI / REDONDEAR


    │   ├── Funciones lógicas: SI / Y / O / NO / SI.ERROR


    │   ├── Funciones de búsqueda: BUSCARV / BUSCARX / ÍNDICE / COINCIDIR


    │   ├── Funciones de texto: CONCAT / MAYUSC / IZQUIERDA / LARGO / SUSTITUIR


    │   └── Funciones de fecha: HOY / AHORA / DIA / MES / AÑO / DIAS


    ├── GRÁFICOS: seleccionar datos → Insertar → tipo de gráfico


    │   Tipos: columnas/barras (comparar) / líneas (tendencia) / circular (proporción) /


    │           dispersión (relación) / combinado


    ├── GESTIÓN DE DATOS


    │   ├── Ordenar: Datos → Ordenar (A→Z / Z→A / multinivel)


    │   ├── Filtrar (Ctrl+Mayúsc+L): muestra solo filas que cumplen criterios


    │   ├── Tablas de Excel (Ctrl+T): fórmulas automáticas + autofiltro + estilos


    │   ├── Tablas dinámicas: Insertar → Tabla dinámica → campos a Filas/Columnas/Valores


    │   ├── Formato condicional: resalta celdas según criterios (colores, barras, iconos)


    │   └── Validación de datos: restringe qué datos pueden introducirse en una celda


    └── Pegado especial (Ctrl+Alt+V): Todo / Fórmulas / VALORES / Formatos / Transponer
