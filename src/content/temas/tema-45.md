│


├── SISTEMA RETRIBUTIVO PÚBLICO


│   ├── Funcionarios: TREBEP / Ley de Presupuestos / normativa de cada AAPP


│   └── Personal laboral: ET + convenio colectivo


│


├── RETRIBUCIONES DE FUNCIONARIOS


│   ├── BÁSICAS (iguales para todos los del mismo subgrupo)


│   │   ├── Sueldo: según Grupo/Subgrupo (A1/A2/C1/C2/Agrupaciones)


│   │   │   → C1: Bachillerato — el subgrupo de las oposiciones que preparamos


│   │   ├── Trienios: cuantía fija por cada 3 años / importe varía por grupo


│   │   └── Pagas extraordinarias: 2 al año (junio y diciembre) = 1 mensualidad (sueldo + trienios)


│   │


│   └── COMPLEMENTARIAS (específicas del puesto y del funcionario)


│       ├── Complemento de destino: retribuye el NIVEL del puesto (del 1 al 30)


│       │   → Igual para todos los funcionarios del mismo nivel


│       ├── Complemento específico: condiciones particulares del puesto


│       │   (dificultad técnica / dedicación / incompatibilidad / responsabilidad / peligrosidad)


│       │   → Variable por puesto / fijado en la RPT


│       ├── Complemento de productividad: rendimiento individual / variable / no consolida


│       ├── Gratificaciones servicios extraordinarios: horas fuera de jornada / excepcionales


│       └── Carrera profesional horizontal: grados/escalones sin ascender de puesto


│


├── RETRIBUCIONES PERSONAL LABORAL


│   └── Salario base + complementos salariales + pagas extras


│       → Fijadas en convenio colectivo (AGE: IV Convenio Único / CyL: convenio propio)


│


├── LA NÓMINA


│   ├── Documento contable (O+P) + laboral + tributario


│   ├── DEVENGOS (percepciones salariales + no salariales):


│   │   Sueldo / Trienios / C. Destino / C. Específico / C. Productividad /


│   │   Gratificaciones / Prorrata pagas extras / Dietas e indemnizaciones


│   ├── DEDUCCIONES:


│   │   ├── Cotización SS trabajador: contingencias comunes (4,70%) + desempleo (1,55%) +


│   │   │   FP (0,10%) + MEI (0,13%)


│   │   ├── Retención IRPF: calculada sobre retribuciones anuales y circunstancias personales


│   │   │   → Empleado comunica datos con modelo 145


│   │   │   → Administración ingresa retenciones con modelo 111


│   │   │   → Certificado de retenciones al cierre del año


│   │   └── Otras: cuotas sindicales / anticipos / embargos / reintegros


│   └── LÍQUIDO = Total devengos − deducciones


│


├── ALTAS Y BAJAS EN NÓMINA


│   ├── Alta: toma de posesión / reingreso → comunicar a SS antes del inicio


│   ├── Baja: cese / jubilación / excedencia sin retribuciones → comunicar en 3 días


│   └── IT: subsidio (60% días 4-20 / 75% día 21+) — AAPP suele complementar al 100%


│


├── SEGURIDAD SOCIAL DEL PERSONAL LABORAL


│   ├── Régimen General SS: mismo que empresa privada


│   ├── Acción protectora: IT / IP / jubilación / muerte y supervivencia /


│   │   maternidad-paternidad / desempleo / riesgo embarazo


│   ├── Bases de cotización: mínimas y máximas fijadas en Ley de Presupuestos


│   └── Gestión: Sistema RED (telemático) — altas, bajas, partes de cotización (TC2)


│


└── MUFACE Y CLASES PASIVAS


    ├── MUFACE: funcionarios civiles AGE


    │   → Asistencia sanitaria: elección libre entre red pública o aseguradora privada


    │   → Prestaciones IT / IP / fallecimiento


    │   → Funcionarios locales y autonómicos: RGSS (regla general)


    │   → CyL: funcionarios Junta → RGSS


    ├── Clases Pasivas: pensiones de funcionarios AGE ingresados ANTES del 1/1/2011


    │   → Jubilación / viudedad / orfandad / inutilidad


    │   → Gestionadas por INSS desde 2011


    └── Desde 1/1/2011: nuevos funcionarios AGE → RGSS (convergencia con sistema general)


        → Los ingresados antes de 2011 mantienen Clases Pasivas
