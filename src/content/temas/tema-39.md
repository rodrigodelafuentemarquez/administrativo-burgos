│


├── ACCESS 365 — BASE DE DATOS RELACIONAL


│   ├── Objetos: TABLAS / CONSULTAS / FORMULARIOS / INFORMES / Macros / Módulos


│   ├── Archivo: extensión .accdb


│   │


│   ├── TABLAS


│   │   ├── Registros (filas) + Campos (columnas)


│   │   ├── Tipos de datos: Texto corto / Texto largo / Número / Fecha-Hora /


│   │   │                   Moneda / Autonumérico / Sí-No / Calculado / Hipervínculo


│   │   ├── Clave principal: identifica cada registro de forma única e inequívoca


│   │   │   → Autonumérico: opción más habitual para clave principal


│   │   ├── Vista Hoja de datos (ver/editar datos) ↔ Vista Diseño (estructura/campos)


│   │   └── Propiedades del campo: tamaño / formato / máscara / regla de validación /


│   │                              valor predeterminado / requerido / indexado


│   │


│   ├── RELACIONES (Herramientas BD → Relaciones)


│   │   ├── 1:1 — uno a uno


│   │   ├── 1:N — uno a varios (la más habitual)


│   │   └── M:N — varios a varios → necesita tabla intermedia


│   │   Integridad referencial: evita datos huérfanos / opción actualizar/eliminar en cascada


│   │


│   ├── CONSULTAS


│   │   ├── Selección: recupera y muestra datos según criterios (no modifica los datos)


│   │   ├── Parámetros: solicita valor al usuario al ejecutarse


│   │   ├── Totales: agrupa y calcula (suma, promedio, máximo, mínimo, recuento)


│   │   ├── Tabla de referencias cruzadas: filas × columnas (como tabla dinámica)


│   │   └── Acción: modifica datos → datos anexados / eliminación / actualización / creación de tabla


│   │   Diseño QBE: cuadrícula visual + criterios (misma fila=Y / filas distintas=O)


│   │   → Access genera SQL automáticamente


│   │


│   ├── FORMULARIOS


│   │   ├── Para introducir, ver y editar datos de forma visual


│   │   ├── Creación: Asistente / Automático / Vista Diseño / Formulario dividido


│   │   ├── Controles: cuadro de texto / etiqueta / combo / lista / casilla verificación /


│   │   │             botón de comando / subformulario


│   │   └── Secciones: Encabezado del formulario / Detalle / Pie del formulario


│   │


│   ├── INFORMES


│   │   ├── Para presentar datos formateados, listos para imprimir o exportar


│   │   ├── Agrupación + totales por grupo y totales generales


│   │   └── Secciones: Encabezado del informe / Encabezado de página /


│   │                   Encabezado de grupo / Detalle / Pie de grupo /


│   │                   Pie de página / Pie del informe


│   │


│   └── IMPORTACIÓN, VINCULACIÓN Y EXPORTACIÓN (Datos externos)


│       ├── Importación: copia datos del origen a Access (independiente del origen)


│       │   → Desde: Excel / CSV-TXT / XML / otras BD


│       ├── Vinculación: Access accede a datos externos sin copiarlos (se actualizan en tiempo real)


│       └── Exportación: envía datos de Access a Excel / PDF / CSV / XML / Word


│


└── OUTLOOK 365 — CORREO ELECTRÓNICO Y PRODUCTIVIDAD


    ├── Secciones: Correo / Calendario / Personas (Contactos) / Tareas / Notas


    ├── Carpetas predeterminadas: Bandeja de entrada / Borradores / Enviados /


    │                             Eliminados / Correo no deseado / Archivo


    │


    ├── ENVIAR MENSAJES


    │   ├── Campos: Para (destinatario) / CC (copia visible) / CCO (copia oculta, privacidad)


    │   │   → En AAPP: usar CCO en envíos masivos para proteger datos personales (RGPD)


    │   ├── Asunto: resumen del mensaje / Cuerpo: texto con formato enriquecido


    │   └── Adjuntar archivos + opciones: importancia / confirmación lectura / retraso en envío


    │


    ├── RECIBIR / RESPONDER / REENVIAR


    │   ├── Responder (Ctrl+R): solo al remitente


    │   ├── Responder a todos (Ctrl+Mayúsc+R): al remitente y todos los destinatarios


    │   └── Reenviar (Ctrl+M): enviar el mensaje a nuevos destinatarios


    │


    ├── REGLAS DE MENSAJE (Inicio → Reglas → Administrar reglas y alertas)


    │   ├── Estructura: CONDICIÓN → ACCIÓN (→ Excepción)


    │   ├── Condiciones: remitente / asunto / destinatario / tamaño / importancia


    │   ├── Acciones: mover / copiar / eliminar / reenviar / categorizar / responder auto


    │   └── Respuestas automáticas / fuera de la oficina: Archivo → Respuestas automáticas


    │


    └── LIBRETA DE DIRECCIONES Y CONTACTOS


        ├── Libreta de direcciones: almacén de direcciones de correo y datos de contacto


        ├── Lista global de direcciones (GAL): todos los usuarios de la organización (Exchange/M365)


        ├── Sección Personas: contactos completos (nombre, email, teléfono, dirección…)


        ├── Grupos de contactos (listas de distribución): envío a múltiples destinatarios con un solo nombre


        └── Autocompletar: sugerencias de direcciones al escribir en Para/CC/CCO
